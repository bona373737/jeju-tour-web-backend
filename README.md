# jeju-tour-web-backend
- 제주도 여행정보(관광지,숙박,음식점)제공 웹사이트

#### [ 프로젝트 기술스택 및 개발환경 ]
- Frontend : HTML, CSS, SCSS, JS, ReactJS, Redux-toolkit
- Backend :  NodeJS, Express, MyBatis
- DB : MySQL
- 테스트서버 : ubuntu 개인서버컴퓨터 구축
- 배포서버 : AWS EC2
